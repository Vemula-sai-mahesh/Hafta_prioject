Setup & Installation

Being in beta - and not consumer ready
Please follow the following steps


Frontend Setup

  • cd .\front
  • npm run install
  • npm run dev

Backend Setup

  • pip install -r requirements.txt
  • python run.py

Config

Hafta currently runs using SQLAlcehmy, running on MySQL

Set the database URI in config.py
